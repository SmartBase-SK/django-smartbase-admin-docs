# Icons

Use the exact icon name as the value of the `icon` parameter.

```python
SBAdminMenuItem(view_id="dashboard", icon="All-application")
SBAdminCustomAction(title="Export to XLSX", icon="Excel-one", ...)
```

Icon names now match the SVG filename without the `.svg` extension, so use the exact kebab-cased value shown in the catalog, for example `Pin-filled`, `Sort-alt`, or `Application-filled`.
